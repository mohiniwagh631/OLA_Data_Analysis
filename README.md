🚖 Ola Ride Booking Data Analysis Dashboard
📌 Overview

This project focuses on analyzing Ola ride booking data to gain insights into ride operations, customer behavior, revenue generation, and service performance. Using SQL for data exploration and Power BI for visualization, the dashboard transforms raw booking data into meaningful business intelligence.

The analysis is based on a dataset containing over 100,000 ride records and provides a comprehensive view of booking trends, cancellations, customer preferences, and operational efficiency.

🎯 Project Objectives
Monitor ride booking and completion trends
Evaluate cancellation patterns and their impact
Analyze revenue generated from rides
Understand customer and driver behavior
Compare vehicle category performance
Identify preferred payment methods
Support data-driven decision-making through interactive visualizations
